# Daily Quote App (Flutter + GetX + Dio)

A mobile application that displays random quotes, allows users to save favorites, and share quotes.  
This project focuses not only on coding, but also on **effective AI-tool usage** to speed up development and improve code quality.

---

##  Features

- Random quote on home screen  
- Add quotes to favorites  
- View favorite quotes list  
- Share quotes via system share sheet  
- Persistent storage using GetStorage  
- Clean UI with gradient AppBar and styled cards
